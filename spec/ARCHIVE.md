## 一、项目概述 (来自 SPEC.md)

### 1.1 项目背景

在现有 OKX 量化交易系统的基础上，增加手动交易功能模块，同时集成大语言模型（LLM）API，为用户的手动交易提供智能分析、风险提醒和决策支持。

### 1.2 项目目标

- 提供完整的手动交易功能界面和 API
- 集成大语言模型进行交易分析和风险提醒
- 结合技术指标、新闻事件、宏观经济数据进行综合分析
- 在用户进行手动交易时提供实时提醒和建议

### 1.3 核心功能模块

| 模块 | 路径 | 说明 |
|------|------|------|
| 手动交易模块 | internal/manualtrading/ | manager.go, order_manager.go, position_manager.go |
| 大模型分析模块 | internal/llmanalysis/ | client.go, providers/, analyzer.go, prompts.go |
| 数据采集模块 | internal/dataservice/ | 新闻、经济数据采集 |
| 提醒服务模块 | internal/alertservice/ | 提醒引擎、通知发送 |
| 数据存储模块 | internal/storage/ | SQLite 数据库、Repository |

### 1.4 API 设计

| 端点 | 方法 | 说明 |
|------|------|------|
| /api/manual/order | POST | 创建订单 |
| /api/manual/order/{order_id} | DELETE | 撤销订单 |
| /api/manual/orders | GET | 查询订单 |
| /api/manual/position/close | POST | 平仓 |
| /api/manual/position/tp-sl | POST | 设置止盈止损 |
| /api/llm/analyze/trade | POST | 交易分析 |
| /api/llm/analyze/positions | GET | 持仓分析 |
| /api/llm/analyze/market | GET | 市场概览 |
| /api/data/news | GET | 获取新闻 |
